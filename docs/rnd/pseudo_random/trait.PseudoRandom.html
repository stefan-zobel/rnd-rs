Expand description

A generator of uniform pseudorandom values.

Implementors need to supply an implementation of the next_long method.

The default implementations in this trait are efficient for methods that need at least 33 bits of randomness but somehow wasteful for the other methods because it dissipates valuable random bits piled up in the call to next_long() whenever less than 33 random bits are needed for the result type.
